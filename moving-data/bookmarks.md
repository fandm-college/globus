# Using Bookmarks

Bookmarks save time by allowing you to return directly to frequently used locations without searching for the collection each time.

Unlike a web browser bookmark, a Globus bookmark remembers **both the collection and the current folder within that collection**.

## Create a bookmark

Navigate to the folder you want to save. Click the bookmark ribbon to the right of the **Path** field.

![Annotated screenshot showing the bookmark ribbon icon]({{ site.baseurl }}/assets/images/annotated/bookmark-ribbon.png){: .figure }

<div class="figure-caption"><strong>Figure 10.</strong> Click the ribbon icon to bookmark the current location.</div>

Enter a descriptive name and click **Create Bookmark**.

![Annotated screenshot showing bookmark name dialog]({{ site.baseurl }}/assets/images/annotated/bookmark-name-dialog.png){: .figure }

<div class="figure-caption"><strong>Figure 11.</strong> Name the bookmark, then click Create Bookmark.</div>

After the bookmark is created, the ribbon is filled in yellow.

![Annotated screenshot showing created bookmark]({{ site.baseurl }}/assets/images/annotated/bookmark-created.png){: .figure }

<div class="figure-caption"><strong>Figure 12.</strong> The filled yellow ribbon indicates the current location has been bookmarked.</div>

## Use bookmarks

When searching for a collection, select the **Bookmarks** tab to view saved locations.

![Annotated screenshot showing Bookmarks tab]({{ site.baseurl }}/assets/images/annotated/bookmarks-tab.png){: .figure }

<div class="figure-caption"><strong>Figure 13.</strong> Use the Bookmarks tab to quickly access saved locations.</div>

{: .good-to-know }
> Bookmarks are personal. They are stored with your Globus account, available from any computer where you log in, and not visible to collaborators.

You can rename a bookmark using the pencil icon or remove it using the trash can icon.

{: .recommendation }
> If you regularly transfer data between the same source and destination, create bookmarks for both locations.
