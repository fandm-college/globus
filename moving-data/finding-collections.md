# Finding and Accessing Collections

## At a glance

- Search for collections by name, institution, or project.
- Use collection links when collaborators share data with you.
- Some collections require first-time authorization.
- Bookmark frequently used locations.

Before transferring data, locate the storage locations involved in the transfer.

Click inside either **Collection** search box to open Collection Search.

![Collection search dialog in Globus]({{ site.baseurl }}/assets/images/screenshots/collection-search.png){: .figure }

<div class="figure-caption"><strong>Figure 6.</strong> Collection Search lets you find collections from Recent, Bookmarks, Your Collections, or Shared With You.</div>

## Search tips

If you cannot find a collection:

- Try fewer search terms.
- Search by institution rather than project name.
- Confirm that you have been granted access.
- If someone shared data with you, ask them to send the **collection link**.

## First-time access

The first time you access some collections, Globus may ask you to authorize access before browsing or transferring files.

{: .good-to-know }
> Seeing an authorization or consent screen the first time you access a collection is expected behavior. It does not mean anything is wrong.

## Need a collection or research storage?

If you need a location to store research data or would like a Globus Collection created for your research project, contact Research Computing Services through [request.fandm.edu](https://request.fandm.edu). Search for **Research Computing** and select any Research Computing request option.
