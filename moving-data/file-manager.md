# Understanding the File Manager

## At a glance

- The File Manager is where you browse collections and transfer files.
- Use the **two-panel view** for most transfers.
- Either panel can be the source or destination.
- Transfers begin from the panel where you click **Start**.

## What is a collection?

A **Collection** is a storage location that Globus can access. A collection might represent a campus storage system, an HPC cluster, a laboratory instrument, a collaborator's shared data, or your own computer using Globus Connect Personal.

Every transfer moves data from one collection to another.

{: .good-to-know }
> Older Globus documentation may refer to collections as **endpoints**. This guide uses the current Globus terminology: **Collection**.

## Use the two-panel view

When the File Manager first opens, it may display a single panel. For transfers, Research Computing Services recommends switching to the two-panel view.

The two-panel view displays two collections side by side.

![Globus File Manager in two-panel view]({{ site.baseurl }}/assets/images/screenshots/file-manager-two-pane.png){: .figure }

<div class="figure-caption"><strong>Figure 5.</strong> The Globus File Manager in two-panel view.</div>

{: .important }
> Neither panel is permanently assigned as the source or destination. The **source** is the panel where you click **Start**. The **destination** is the opposite panel.

{: .recommendation }
> Use the two-panel layout whenever transferring files. Seeing both collections at the same time makes it easier to verify that you are copying data to the intended destination.
