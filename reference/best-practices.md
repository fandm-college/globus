# Best Practices

## Organizing data

- Use descriptive folder names.
- Keep project data together.
- Bookmark frequently used locations.
- Archive completed projects when appropriate.

## Sharing

- Share the smallest directory necessary.
- Verify collaborator Identity IDs or Group UUIDs.
- Prefer Groups over many individual shares.
- Remove collaborators when access is no longer needed.
- Use read-only access unless write access is required.

## Transfers

- Prefer Globus transfers over browser downloads.
- Use descriptive transfer labels.
- Enable checksum synchronization for recurring transfers.
- Monitor Activity for transfer status.
- Watch for Globus email notifications.

## Timers

- Use timers for recurring transfers.
- Avoid repeating schedules every few minutes.
- Review timers periodically.
- Use descriptive timer names.

## Globus Connect Personal

- Use a descriptive collection name.
- Share only folders you intend to expose.
- Enable Write only when data should be copied to your computer.
- Configure Globus Connect Personal to start automatically.
- Keep your computer powered on and connected when transfers are expected.
