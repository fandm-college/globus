# Troubleshooting

## I cannot find a collection

- Try fewer search terms.
- Use the collection link if someone shared data with you.
- Confirm you have been granted access.
- Approve any required consent screens.
- Contact the collection owner or Research Computing Services.

## My transfer failed

- Check **Activity** for error details.
- Review the Globus email notification.
- Verify source and destination permissions.
- Confirm both collections are available.

## My collaborator cannot see my data

- Verify the correct directory was shared.
- Confirm the collaborator's Identity ID or Group UUID.
- Check read/write permissions.
- Ask the collaborator to open the collection link.

## Globus Connect Personal is offline

- Confirm the application is running.
- Make sure the computer is awake and connected to the internet.
- Confirm you are signed in with the correct Globus account.
- Verify the relevant folder is configured as Shared.

## A timer did not run

- Check the timer schedule.
- Review **Activity**.
- Confirm the source and destination collections are available.
- Verify Globus Connect Personal is running if one of the collections is your computer.

## Browser download is slow or incomplete

Use a Globus transfer instead. Browser downloads do not automatically resume and only support individual files.
