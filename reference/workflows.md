# Common Workflows

These examples show how Globus features fit together in common research workflows.

![Diagram of common Globus workflow]({{ site.baseurl }}/assets/images/diagrams/common-workflows.svg){: .figure }

## Copy data from an instrument to research storage

1. Confirm Research Computing Services has configured the instrument as a Globus Collection.
2. Open the instrument collection.
3. Open the destination research collection.
4. Select the files or directories.
5. Start the transfer from the source pane.
6. Monitor progress in **Activity**.

## Send data to a collaborator

1. Ask the collaborator for their collection link.
2. Open the link in Globus.
3. Select the data to send.
4. Start the transfer.
5. Confirm completion in **Activity**.

## Share data with students or a research group

1. Create a Globus Group.
2. Add students or collaborators to the Group.
3. Share the project directory with the Group.
4. Update the Group membership as people join or leave.

## Work from home

1. Install Globus Connect Personal.
2. Configure targeted shared folders.
3. Transfer between your Personal Collection and research storage.
4. Keep Globus Connect Personal running while transfers are expected.

## Archive old data

1. Identify the project folder to archive.
2. Open the archive destination collection.
3. Transfer the project folder.
4. Verify completion in **Activity**.
