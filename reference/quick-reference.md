# Quick Reference

## Common recommendations

- Use Globus transfers instead of browser downloads.
- Use the two-panel File Manager view.
- Click **Start** in the source panel.
- Monitor transfers in **Activity**.
- Use bookmarks for frequently used folders.
- Verify collaborator Identity IDs.
- Use Groups for labs, courses, and changing teams.
- Share the smallest directory necessary.
- Use read-only permissions unless write access is needed.
- Avoid Public and All Users sharing.
- Use checksum synchronization for recurring transfers.
- Configure Globus Connect Personal to start automatically.

## Common icons and menu items

| Item | Meaning |
|:---|:---|
| **File Manager** | Browse collections and transfer files |
| **Activity** | Monitor transfer progress and history |
| **Groups** | Manage collaboration groups |
| **Timers** | View or manage scheduled transfers |
| **Permissions** | Share data with users or groups |
| **Bookmark ribbon** | Save the current collection and path |
| **Start** | Begin a transfer from the active panel |
