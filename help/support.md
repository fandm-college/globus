# Need Help?

Research Computing Services can assist with:

- New Globus Collections
- Scientific instrument integrations
- Large data migrations
- HPC data movement
- Secure data sharing
- Storage planning
- Transfer Timer assistance
- Globus Connect Personal troubleshooting
- Permission and sharing issues
- General Globus support

## Submit a request

To request assistance:

1. Visit [https://request.fandm.edu](https://request.fandm.edu).
2. Search for **Research Computing**.
3. Select any available Research Computing request type.
4. Submit your request with as much detail as possible.

Your request will be routed to Research Computing Services.

{: .help }
> If you need research storage or would like a Globus Collection created for your research project, submit a ticket through the Service Portal and search for **Research Computing**.

## Include helpful details

When submitting a request, include:

- The collections involved
- Source and destination paths, if applicable
- Transfer task IDs, if available
- Collaborator Identity IDs or Group UUIDs, if relevant
- Screenshots of error messages
- Whether the data is protected or regulated
