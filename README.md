# F&M Globus User Guide

A Just the Docs site for Research Computing Services Globus documentation.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

Open the local URL printed by Jekyll.

## Structure

Related Markdown files are organized into directories:

- `welcome/`
- `moving-data/`
- `collaboration/`
- `automation/`
- `personal-computer/`
- `reference/`
- `help/`

## Screenshot updates

Screenshots are stored in:

- `assets/images/screenshots/`
- `assets/images/annotated/`
- `assets/images/diagrams/`
