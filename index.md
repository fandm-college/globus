Globus is a data management service that facilites data

- Transfer
- Sharing
- Discovery

It provides fast and secure data movement especially for large
amounts of data (100s of GB and more).  It also includes tools
for research workflow automation and data discovery via metadata
search.

For more information you can view other Globus tutorials provided here
or explore more about Globus on the [Globus website](https://www.globus.org/)
